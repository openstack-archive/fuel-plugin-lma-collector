Heka module for Puppet
======================

Description
-----------

Puppet module for configuring Heka.

Usage
-----

To deploy the Heka service on a host.

```puppet
# Configure the base heka service
class {'heka':
}
```

Limitations
-----------

This module supports only Fuel-based deployments using Neutron.
